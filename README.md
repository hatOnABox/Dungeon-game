# Dungeon game
This Github repository is a text-based rpg game written in Python 3.6.4 (https://www.python.org/downloads/release/python-364/). The game loads in a map which helps the player visualize their surroundings and environment (for more details for the maps read the mapKey.txt in the maps folder). The player interacts with objects and fights monsters. The game has classes, items, bosses, and more!

### Future features

* status conditions
* secret bosses
* magic items
